The LCM PowerPoint Accessibility Checker helps identify common accessibility issues in PowerPoint presentations and provides guidance for remediation.

Having trouble?

If you encounter issues loading the add-in, please verify:

  • You are using the desktop or web version of PowerPoint
  • The add-in was installed using a valid manifest
  • You have an active internet connection

For additional help, contact Last Call Media.
